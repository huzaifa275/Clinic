# AuraSmile Dental Clinic Website

A modern, responsive website developed for AuraSmile Dental Clinic with a strong focus on user experience, performance, accessibility, and online appointment booking.

## Features

- Modern responsive design
- AI-powered patient assistant
- Online appointment booking
- Mobile-friendly interface
- Fast loading performance
- SEO optimized
- Social media preview support
- Professional dental services pages
- Contact and location information
- Secure production-ready configuration

## Technology Stack

- React
- TypeScript
- Vite
- Tailwind CSS

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Deployment

This project can be deployed on any modern static hosting platform such as Netlify or Vercel.
